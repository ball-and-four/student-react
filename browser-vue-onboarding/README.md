# browser-javascript-onboarding
javascript 온보딩 과제

## 로컬 개발환경 실행 방법

다음 npm 명령어를 터미널에 입력합니다.

### 1. 설치

이 프로젝트를 실행하기 위해 필요한 패키지들을 설치합니다. 

```shell
npm install
```

### 2. 실행

준비된 script 을 이용하여 로컬 서버를 실행합니다.

```shell
npm run dev
```
